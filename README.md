Local server setup
You'll need a local server to load external files and avoid CORS issues. There are two options:


Python server:
Open your terminal/command prompt
Navigate to your project folder: cd path/to/lab1
If using Python 3: python -m http.server 8000
If using Python 2: python -m SimpleHTTPServer 8000
Open your browser and visit: http://localhost:8000
To stop the server: Press Ctrl+C in the terminal

VS Code Live Server :

Install the "Live Server" extension in VS Code
Remember to click on install.

To run the application run Go Live in VS Code or run the python server
